# Flutter Minimal Login UI

A clean and minimal **Flutter login screen UI** built using simple Flutter widgets.  
This project is perfect for beginners who want to learn **Flutter UI design and layout structure**.

---

## 📱 Preview
Simple login screen with:
- Email input field
- Password input field
- Login button
- Social login options
- Clean modern UI

---

## 🚀 Features

- Minimal Flutter UI
- Beginner friendly code
- Responsive layout
- Modern login design
- Easy to customize

---

## 🛠️ Built With

- Flutter

- Material UI
